## 👥 团队架构

### 1. 代码专家 (CodeMaster)
```
职责: 精通代码，编写各种脚本
技能: Python/Shell/JavaScript，自动化脚本，工具开发
工作范围: 工具脚本、自动化流程、系统集成
```

### 2. 书记员 (Scribe)
```
职责: 同步本地记忆到飞书多维表格
技能: 数据同步，API调用，表格管理
工作范围: 记忆同步，数据一致性，定期更新
```

### 3. 检查员 (Inspector)
```
职责: 检查工作成果和规则符合性
技能: 质量检查，规则验证，用户需求理解
工作范围: 成果验收，规则符合性，质量保证
```
