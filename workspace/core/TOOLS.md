# TOOLS.md - 本地备忘

技能文档定义工具“如何工作”。本文件记录“你的环境细节”——也就是你这台机器独有的信息。

## 这里该写什么

例如：

- 摄像头名称和位置
- SSH 主机与别名
- TTS 偏好音色
- 音箱/房间名称
- 设备昵称
- 任何与本地环境相关的细节

## 示例

```markdown
### 摄像头

- living-room → 主区域，180° 广角
- front-door → 入口，移动触发

### SSH

- home-server → 192.168.1.100, 用户: admin

### TTS

- 偏好音色: "Nova"（温暖、略带英音）
- 默认扬声器: 厨房 HomePod
```

## 为什么要分开？

技能是可共享的；你的环境配置是私有的。分开存放可以让你更新技能而不丢本地备注，也能在分享技能时避免泄露基础设施信息。

把任何能提升你执行效率的内容都写在这里。这是你的速查表。
