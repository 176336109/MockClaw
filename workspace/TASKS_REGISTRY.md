## 📁 文件管理规范

| 事务类型 | 目录结构 | 必需文件 | 可选文件 |
|----------|----------|----------|----------|
| 任务 | `Tasks/TASK_ID_任务名/` | `README.md`, `progress.md` | `artifacts/`, `test_report.json` |
| 工程 | `Projects/ENG_ID_工程名/` | `README.md`, `progress.md` | `Team/`, `deliverables/`, `tasks.md` |

**示例：**
```
Tasks/
└── TASK_20260302_001_测试记忆同步系统/
    ├── README.md          # 任务说明 + 成功标准
    ├── progress.md        # 进度日志（情景记忆）
    └── test_report.json   # 测试报告

Projects/
└── ENG-20260302-小红书/
    ├── README.md          # 需求分析 + 设计文档
    ├── progress.md        # 工程进度（情景记忆）
    ├── tasks.md           # 子任务列表
    └── test_report.json   # 测试报告
```
