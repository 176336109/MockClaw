# Feishu美观文档创建Skill

## 核心功能
1. **创建格式美观的飞书文档**
2. **支持丰富的块类型和样式**
3. **提供预定义模板**
4. **简化复杂API调用**

## 使用场景
- 创建业务方案文档
- 撰写项目报告
- 制作产品文档
- 生成会议纪要
- 编写技术文档

## 快速开始

### 创建简单美观文档
```javascript
// 使用预定义模板
创建美观文档({
  模板: "业务方案",
  标题: "龙虾知识星球运营方案",
  内容: {
    核心摘要: [
      "模式：AI（龙虾）100%自主运营",
      "定价：19元/年（极致低价）",
      "分工：你们收钱监督，我干活运营",
      "目标：10,000用户/年，收入190,000元"
    ],
    四大优势: [
      "全AI运营 · 国内首个AI自主运营知识星球",
      "极致低价 · 19元/年，零决策成本",
      "透明运营 · 公开AI运营数据和逻辑",
      "持续进化 · AI根据反馈自动优化"
    ]
  }
});
```

### 自定义文档结构
```javascript
// 自定义块结构
创建自定义文档({
  标题: "自定义文档",
  块列表: [
    { 类型: "一级标题", 文本: "主标题", 加粗: true },
    { 类型: "分隔线" },
    { 类型: "二级标题", 文本: "章节标题" },
    { 类型: "文本", 文本: "正文内容" },
    { 类型: "无序列表", 文本: "列表项1" },
    { 类型: "无序列表", 文本: "列表项2" },
    { 类型: "待办事项", 文本: "任务1", 完成: false },
    { 类型: "待办事项", 文本: "任务2", 完成: true },
    { 类型: "引用", 文本: "重要说明" },
    { 类型: "代码块", 文本: "console.log('示例')", 语言: "javascript" }
  ]
});
```

## 支持的块类型

### 文本块
| 类型 | 说明 | 示例 |
|------|------|------|
| 文本 | 普通文本 | `{ 类型: "文本", 文本: "内容" }` |
| 加粗文本 | 粗体文本 | `{ 类型: "文本", 文本: "重要", 加粗: true }` |
| 斜体文本 | 斜体文本 | `{ 类型: "文本", 文本: "强调", 斜体: true }` |
| 彩色文本 | 带颜色文本 | `{ 类型: "文本", 文本: "红色", 颜色: "#FF0000" }` |

### 标题块
| 类型 | 说明 | 示例 |
|------|------|------|
| 一级标题 | H1标题 | `{ 类型: "一级标题", 文本: "主标题" }` |
| 二级标题 | H2标题 | `{ 类型: "二级标题", 文本: "章节" }` |
| 三级标题 | H3标题 | `{ 类型: "三级标题", 文本: "小节" }` |

### 列表块
| 类型 | 说明 | 示例 |
|------|------|------|
| 无序列表 | 项目符号列表 | `{ 类型: "无序列表", 文本: "项目" }` |
| 有序列表 | 数字编号列表 | `{ 类型: "有序列表", 文本: "第一项" }` |
| 待办事项 | 复选框列表 | `{ 类型: "待办事项", 文本: "任务", 完成: true }` |

### 特殊块
| 类型 | 说明 | 示例 |
|------|------|------|
| 分隔线 | 水平分隔线 | `{ 类型: "分隔线" }` |
| 引用 | 引用块 | `{ 类型: "引用", 文本: "引用内容" }` |
| 代码块 | 代码片段 | `{ 类型: "代码块", 文本: "代码", 语言: "python" }` |

## 错误处理

### 常见错误
```javascript
try {
  创建美观文档({ /* 参数 */ });
} catch (error) {
  if (error.类型 === "API错误") {
    console.log("飞书API错误:", error.消息);
  } else if (error.类型 === "参数错误") {
    console.log("参数错误:", error.消息);
  } else {
    console.log("未知错误:", error);
  }
}
```

### 重试机制
```javascript
创建文档带重试({
  参数: { /* 文档参数 */ },
  最大重试次数: 3,
  重试间隔: 1000 // 毫秒
});
```

## 集成示例

### 与OpenClaw集成
```javascript
// 在OpenClaw中使用
const 结果 = await 执行工具("feishu_beautiful_doc", {
  action: "create",
  template: "business_plan",
  data: {
    title: "龙虾知识星球运营方案",
    sections: {
      summary: "核心摘要内容",
      advantages: ["优势1", "优势2"],
      business_model: "商业模式描述"
    }
  }
});
```

### 命令行使用
```bash
# 创建文档
openclaw feishu-beautiful-doc create --template business-plan --title "方案标题"

# 使用JSON数据
openclaw feishu-beautiful-doc create --file data.json

# 批量创建
openclaw feishu-beautiful-doc batch --dir ./documents
```
