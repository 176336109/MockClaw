# 简单战斗系统数值设计

## 系统概述
设计一个基于等级成长的RPG战斗系统，包含属性成长、装备系统、技能伤害计算和战斗平衡。

## 一、核心属性设计

### 1. 基础属性
- **力量(STR)**：影响物理攻击力
- **敏捷(AGI)**：影响攻击速度、闪避率
- **智力(INT)**：影响魔法攻击力、魔法值
- **体质(CON)**：影响生命值、物理防御
- **精神(SPI)**：影响魔法防御、魔法恢复

### 2. 衍生属性
- **生命值(HP)** = 基础HP + CON × 成长系数
- **魔法值(MP)** = 基础MP + INT × 成长系数
- **物理攻击力(ATK)** = 基础ATK + STR × 攻击系数
- **魔法攻击力(MATK)** = 基础MATK + INT × 魔法系数
- **物理防御(DEF)** = 基础DEF + CON × 防御系数
- **魔法防御(MDEF)** = 基础MDEF + SPI × 魔防系数
- **攻击速度(ASPD)** = 基础ASPD + AGI × 速度系数
- **闪避率(Dodge)** = AGI × 闪避系数 / (等级 + 100)

## 二、等级成长系统

### 1. 经验值曲线（指数增长）
```
所需经验值 = 基础经验 × (等级)^1.5
等级1-10：平滑过渡，快速升级
等级11-50：稳定成长，每级时间逐渐增加
等级51+：后期成长，需要更多投入
```

### 2. 属性成长公式
```
每级属性成长 = 基础成长 + 职业系数 × 等级
力量成长 = 2 + 0.1 × 等级（战士职业系数0.3）
敏捷成长 = 2 + 0.08 × 等级（刺客职业系数0.4）
智力成长 = 2 + 0.12 × 等级（法师职业系数0.35）
体质成长 = 2 + 0.15 × 等级（坦克职业系数0.5）
精神成长 = 2 + 0.1 × 等级（牧师职业系数0.4）
```

## 三、战斗伤害公式

### 1. 物理伤害计算
```
基础伤害 = ATK × 技能系数 × (1 + 暴击加成)
实际伤害 = 基础伤害 × (1 - DEF / (DEF + 100 × 等级))
暴击判定：基础暴击率5% + AGI×0.01%
暴击伤害：150% + 暴击伤害加成
```

### 2. 魔法伤害计算
```
基础伤害 = MATK × 技能系数 × 元素克制
实际伤害 = 基础伤害 × (1 - MDEF / (MDEF + 80 × 等级))
元素克制：克制+20%，被克-20%，无关系100%
```

### 3. 命中与闪避
```
命中率 = 基础命中95% + 等级差×1% - 目标闪避率
最小命中率：70%（保证基础命中）
最大命中率：100%（上限）
```

## 四、装备系统设计

### 1. 装备品质与属性
- **普通(白色)**：基础属性
- **优秀(绿色)**：基础属性+1条附加属性
- **稀有(蓝色)**：基础属性+2条附加属性
- **史诗(紫色)**：基础属性+3条附加属性
- **传说(橙色)**：基础属性+4条附加属性+特殊效果

### 2. 装备强化系统
```
强化等级：0-15级
成功率：初始90%，每级降低5%（最低30%）
强化属性：每级提升装备基础属性5%
失败惩罚：0-10级无惩罚，11-15级失败降1级
```

### 3. 装备掉落概率
```
普通怪物：白色80%，绿色15%，蓝色5%
精英怪物：白色50%，绿色30%，蓝色15%，紫色5%
BOSS：白色20%，绿色30%，蓝色30%，紫色15%，橙色5%
```

## 五、技能系统设计

### 1. 技能等级与效果
```
技能等级：1-10级
每级效果提升：10%（线性增长）
学习要求：等级、前置技能、技能点
```

### 2. 技能消耗与冷却
```
魔法消耗 = 基础消耗 × (1 + 0.1 × 技能等级)
冷却时间 = 基础冷却 × (1 - 冷却缩减)
冷却缩减上限：40%
```

### 3. 技能伤害系数示例
```
普通攻击：100% ATK
重击：150% ATK，冷却5秒
火球术：120% MATK，消耗30MP，冷却3秒
治疗术：恢复80% MATK HP，消耗40MP，冷却8秒
```

## 六、平衡性设计

### 1. 职业平衡系数
```
战士：HP+20%，DEF+15%，ATK+10%
刺客：ASPD+25%，暴击率+10%，Dodge+15%
法师：MATK+20%，MP+15%，技能伤害+10%
坦克：HP+30%，DEF+20%，被治疗+15%
牧师：治疗+25%，MDEF+15%，MP恢复+20%
```

### 2. PVP平衡调整
```
PVP伤害修正：70%（降低秒杀可能）
治疗衰减：PVP中治疗效果降低30%
控制时间：PVP中控制效果持续时间减半
```

### 3. 难度曲线设计
```
普通难度：怪物属性=玩家属性×80%
困难难度：怪物属性=玩家属性×120%
史诗难度：怪物属性=玩家属性×180%
怪物技能：随难度增加更多技能和机制
```

## 八、设计原则应用

### 1. 指数增长与对数感知
- 等级经验指数增长，但每级所需时间感知增长平缓
- 属性数值指数提升，但实际战斗效果感知对数变化

### 2. 边际效用递减
- 装备强化收益递减：+10到+11提升小于+0到+1
- 属性堆叠收益递减：单一属性堆叠效果逐渐降低

### 3. 风险与回报平衡
- 高难度副本：高风险（容易死亡），高回报（更好掉落）
- 装备强化：高风险（可能失败），高回报（属性提升）
