## 三、权限需求清单

### 3.1 最小权限集（只读）
- `sheet:spreadsheet:readonly` - 电子表格只读
- `bitable:app:readonly` - 多维表格只读
- `drive:file:readonly` - 文件只读（用于访问表格文件）

### 3.2 完整权限集（读写）
- `sheet:spreadsheet` - 电子表格读写
- `bitable:app` - 多维表格读写
- `drive:file` - 文件读写
- `drive:export:readonly` - 导出权限（可选）

### 3.3 高级功能权限
- `sheet:spreadsheet.formula` - 公式计算（如需要）
- `bitable:app.automation` - 自动化规则（如需要）
- `drive:file.webhook` - Webhook权限（如需要）
