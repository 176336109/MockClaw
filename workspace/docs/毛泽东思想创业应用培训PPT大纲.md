## 幻灯片19：工作流程1-实事求是决策流程
```
提出假设 → 设计实验 → 收集数据 → 
分析数据 → 做出决策 → 记录依据 → 
设置验证 → 循环优化
```

**关键点**：每个决策必须有验证计划

## 幻灯片20：工作流程2-群众路线产品迭代
```
收集反馈 → 分类整理 → 优先级排序 → 
设计方案 → 用户验证 → 开发实现 → 
用户测试 → 发布上线 → 收集新反馈
```

**关键点**：用户全程参与

## 幻灯片21：工作流程3-资源聚焦管理
```
列出项目 → 评估价值 → 计算ROI → 
选择Top 3 → 分配资源 → 设置目标 → 
每周检查 → 每月复盘 → 季度调整
```

**关键点**：敢于说不，聚焦核心
